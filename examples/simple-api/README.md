# Simple API Example

Пример минимального проекта для тестирования системы.

## Идея

"Простой API который возвращает hello world и текущее время"

## Структура

```
simple-api/
├── manifest.yaml           # Манифест проекта
├── constitution.md         # Правила проекта
└── contracts/
    ├── setup.yaml          # Контракт блока setup
    └── endpoints.yaml      # Контракт блока endpoints
```

## Блоки

```
Level 1:  [setup]
             ↓
Level 2:  [endpoints]
```

Всего 2 блока, минимальная сложность для проверки flow.
