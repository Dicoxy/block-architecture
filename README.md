# Block Architecture

Система агентов для AI-driven разработки.

## Идея

Не методология которую читает человек. Операционная система которая превращает идею в работающий продукт через иерархию AI-агентов.

## Как это работает

```
Человек (идея) → Агент-архитектор → Валидация → Агенты уровней → Результат
```

**Человек** — даёт идею, валидирует план. Не пишет код, не составляет детальный план.

**Агент-архитектор** — исследует рынок, декомпозирует идею на уровни, создаёт мини-агентов.

**Агенты уровней** — работают параллельно, каждый отвечает за свой блок, проверяют результаты предыдущего уровня.

## Принципы

1. **Изоляция** — агент знает только свой уровень
2. **Контракты** — между уровнями чёткие форматы входа/выхода
3. **Параллелизм** — уровни работают одновременно где возможно
4. **Валидация человеком** — только на этапе плана, не кода

## Структура

```
/system      — как работают агенты
/contracts   — форматы взаимодействия
/research    — база знаний для агента-архитектора
/logs        — история развития идеи
```

## Цель

Обычный человек + эта система = уровень продвинутого tech lead.

10 человек → миллиардный оборот.
